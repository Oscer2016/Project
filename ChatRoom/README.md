# 聊天室


## 项目综述
     　　为了巩固所学的LinuxC网络编程，我们开发了聊天室项目，该聊天室支持私聊，群聊，FTP，查看消息记录，修改密码以及其他功能。本
     项目使用LinuxC语言和GTK+2.0开发，采用客户端/服务器(C/S)架构，客户端提供一个图形界面，供用户直接操作，操作简单方便，服务器
     和客户端通过网络连接起来，构成一个完整的网络平台。
