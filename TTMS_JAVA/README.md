
# 剧院票务管理系统

## 项目简介
该项目主要功能有演出厅管理、演出计划管理、剧目管理以及出票等,并提供了图形界面进行管理。

## 开发环境
操作系统：Linux(Fedora 26)
开发工具: Eclipse
数据库: MySQL 5.7.19
